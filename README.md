# spot-AI

Starfleet cluster repo for Spot, Daystrom, M-5, and ReadyRoom.

## Layout

- `fleet-docs/` - architecture, hardware, and network documentation
- `nodes/` - per-node notes, configs, backups, and rebuild instructions
- `services/` - systemd units and docker assets
- `configs/` - gateway, worker, and scheduler configs
- `scripts/` - fleet utility and restore scripts
- `manifests/` - package lists, model inventories, and hardware manifests
